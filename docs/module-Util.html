(static) throttle(callback, wait)

Description:
  • 节流，在 wait 毫秒内最多执行 callback 一次
    1. 在函数需要频繁触发时，函数执行一次后，只有大于设定的执行周期后才会执行第二次。
    2. 适合多次事件按时间做平均分配。
    场景：
    1. 窗口调整；
    2. 页面滚动；
    3. DOM元素的拖拽功能实现；
    4. 抢购疯狂点击
